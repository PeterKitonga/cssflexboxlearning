Vertical Centering

Next >>

In this lesson, I learnt how to align items vertically using align-items: center;. Without using flexbox we would use a combination of top: 50%; and transform: translateY(50%); to vertically center an element.

HTML
  1. <section class="vertical-center">
  2.     <div class="vertical-center-item">
  3.         <p>
  4.             Lorem, ipsum dolor sit amet consectetur adipisicing elit. Beatae, ut?
  5.         </p>
  6.     </div><!-- flex item -->
  7.     <div class="vertical-center-item">
  8.         <p>
  9.             Lorem ipsum dolor sit amet, consectetur adipisicing elit. Impedit, atque?
  10.         </p>
  11.     </div><!-- flex item -->
  12. </section>
CSS
  1. .vertical-center {
  2.    display: flex;
  3.    height: 200px;
  4.    /*
  5.    * Using both align-items and justify-content as 'center'
  6.    * will align the flex items in the exact center of the flex container
  7.    */
  8.    align-items: center;
  9.    justify-content: center;
  10.    border: 1px solid rgba(247, 178, 166, 0.8);
  11. }
  12.  
  13. .vertical-center-item {
  14.    border: 1px solid rgba(11, 177, 8, 0.8);
  15.    padding: 10px;
  16. }
  17.  
  18. .vertical-center p {
  19.    margin: 0;
  20. }
