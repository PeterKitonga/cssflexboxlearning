Flex Items: flex-grow

Next >>

In this lesson, I learnt about theflex-growproperty which defines the amount of space a given flex item will take up in the flex container.

HTML
  1. <section class="container-custom">
  2.     <div>1</div><!-- flex item -->
  3.     <div>2</div><!-- flex item -->
  4.     <div>3</div><!-- flex item -->
  5.     <div>4</div><!-- flex item -->
  6. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3.    flex-flow: row nowrap; /* flex-direction and flex-wrap combined */
  4.    justify-content: center;
  5.    align-items: center;
  6.    border: solid 1px green;
  7. }
  8.  
  9. .container-custom div {
  10.    width: 100px;
  11.    /*
  12.    * This property dictates the amount of available space a flex item wound take in the flex container.
  13.    * The higher the value the more space the item will take.
  14.    * More info: https://css-tricks.com/almanac/properties/f/flex-grow/
  15.    */
  16.    flex-grow: 1;
  17. }
  18.  
  19. .container-custom div:nth-child(2n + 1) {
  20.    background-color: darkblue;
  21.    flex-grow: 2;
  22. }
