Flex Containers: flex-wrap

Next >>

In this lesson, I learnt about theflex-wrapproperty which defines whether the flex items are forced in a line or wrapped into multiple lines.

HTML
  1. <section class="container-custom">
  2.     <div>1</div><!-- flex item -->
  3.     <div>2</div><!-- flex item -->
  4.     <div>3</div><!-- flex item -->
  5.     <div>4</div><!-- flex item -->
  6.     <div>5</div><!-- flex item -->
  7.     <div>6</div><!-- flex item -->
  8.     <div>7</div><!-- flex item -->
  9.     <div>8</div><!-- flex item -->
  10.     <div>9</div><!-- flex item -->
  11.     <div>10</div><!-- flex item -->
  12. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3.    flex-direction: row;
  4.    justify-content: flex-start;
  5.    align-items: center;
  6.    /*
  7.    * This property defines whether the flex items are forced in a line or wrapped into multiple lines. 
  8.    * Values include: nowrap(default) | wrap | wrap-reverse
  9.    * This property can be combined with the flex-direction property into one 'flex-flow'. e.g flex-flow: row nowrap;
  10.    * More info: https://css-tricks.com/almanac/properties/f/flex-wrap/
  11.    */
  12.    flex-wrap: wrap;
  13.    border: solid 1px green;
  14.    height: 200px;
  15. }
  16.  
  17. .container-custom div {
  18.    width: 240px;
  19. }
