Flex Containers: flex-direction

Next >>

In this lesson, I learnt about the flex-direction property. This property is used to control the direction of flex items along the main axis of the flex container.

HTML
  1. <section class="container-custom"><!-- flex container -->
  2.     <div></div><!-- flex item -->
  3.     <div></div><!-- flex item -->
  4.     <div></div><!-- flex item -->
  5.     <div></div><!-- flex item -->
  6. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3.    /*
  4.    * This property defines the direction of main axis of the flex container.
  5.    * This inturn affects how the flex items are placed in the main axis.
  6.    * Values include: row | row-reverse | column | column-reverse.
  7.    * (row) values define the main axis horizontally and (column) values
  8.    * vertically.
  9.    * The (column) values can be used to define the overall layout of the
  10.    * webpage and (row) values can define the layout of each section of a webpage.
  11.    */
  12.    flex-direction: column;
  13. }
