Flex Items: order

Next >>

In this lesson, I learnt about theorderproperty which changes the order in which flex items are arranged in a flex container.

HTML
  1. <section class="container-custom">
  2.     <div>1</div><!-- flex item -->
  3.     <div>2</div><!-- flex item -->
  4.     <div>3</div><!-- flex item -->
  5.     <div>4</div><!-- flex item -->
  6.     <div>5</div><!-- flex item -->
  7.     <div>6</div><!-- flex item -->
  8.     <div>7</div><!-- flex item -->
  9.     <div>8</div><!-- flex item -->
  10.     <div>9</div><!-- flex item -->
  11.     <div>10</div><!-- flex item -->
  12.     <div>11</div><!-- flex item -->
  13.     <div>12</div><!-- flex item -->
  14. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3.    flex-flow: row wrap; /* flex-direction and flex-wrap combined */
  4.    justify-content: space-evenly;
  5.    align-items: flex-start;
  6.    align-content: center;
  7.    border: solid 1px green;
  8.    height: 200px;
  9. }
  10.  
  11. .container-custom div {
  12.    width: 240px;
  13.    /*
  14.    * This property changes order in which flex items are displayed. 
  15.    * The larger the number the further the item will be placed.
  16.    * More info: https://css-tricks.com/almanac/properties/o/order/
  17.    */
  18.    order: 0;
  19. }
  20.  
  21. .container-custom div:nth-child(7) {
  22.    background-color: darkblue;
  23.    order: -1;
  24. }
  25.  
  26. .container-custom div:nth-child(2) {
  27.    background-color: darkblue;
  28.    order: 1;
  29. }
  30.  
  31. .container-custom div:nth-child(5) {
  32.    background-color: darkblue;
  33.    order: 2;
  34. }
