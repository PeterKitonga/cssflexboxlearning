In this lesson, I was able to practice on what I have learnt so far to create a responsive layout despite the number of elements present in a row(.grid). The flex items(.cells) respond to the amount of space left for them and take up equal distribution for an evenly spaced out layout. I even added a class .large to increase the flex-grow setting of an element.
