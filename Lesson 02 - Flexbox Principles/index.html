In this lesson, I learnt about the various terminologies and principles of flexbox.

Overview

In the image above we get to see how flexbox works. Items in flexbox are laid out following either the main axis (from main-start to main-end) or the cross axis (from cross-start to cross-end). Here we see that we have various components namely:

  1. Main Axis: This is the primary axis along which flex items are laid out. This depends on the flex-direction property. By default the main axis is horizontal
  2. Main start | Main end: Defines how flex items are placed in the flex container starting from main-start and going to main-end.
  3. Main size: This is the size(width/height) of flex items in the main dimension.
  4. Cross Axis: This axis is perpendicular to the main axis. Its direction depends on the main axis direction.
  5. Cross start | Cross end: Flex lines are filled with items and placed into the container starting on the cross-start side of the flex container and going toward the cross-end side.
  6. Cross size: This is the size(width/height) of flex items in the cross dimension.

Example

HTML
  1. <section class="container-custom"><!-- flex container -->
  2.     <div></div><!-- flex item -->
  3.     <div></div><!-- flex item -->
  4.     <div></div><!-- flex item -->
  5.     <div></div><!-- flex item -->
  6. </section>

  2.    display: flex; /* By default the main axis is horizontal and cross axis is vertical */
