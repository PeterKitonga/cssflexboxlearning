Flex Items: flex-basis

Next >>

In this lesson, I learnt about theflex-basisproperty which defines the base size of flex items before space is allocated. When used there is no need to define a width/height of the flex item. It defines the height when used with flex-direction: column|column-reverse;

HTML
  1. <section class="container-custom">
  2.     <div>1</div><!-- flex item -->
  3.     <div>2</div><!-- flex item -->
  4.     <div>3</div><!-- flex item -->
  5.     <div>4</div><!-- flex item -->
  6. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3.    flex-flow: row nowrap; /* flex-direction and flex-wrap combined */
  4.    justify-content: center;
  5.    align-items: center;
  6.    border: solid 1px green;
  7.    width: 800px; /* let's define a width for the flex container */
  8. }
  9.  
  10. .container-custom div {
  11.    /*
  12.    * This specifies the base size of a flex item before space is allocated depending on the set values for them.
  13.    * The default value is set to 0. Negative values passed are invalid.
  14.    * More info: https://css-tricks.com/almanac/properties/f/flex-basis/
  15.    *
  16.    * Often this property is used together with flex-grow and flex shrink. For that there is a shorthand property
  17.    * like so flex: (grow value) (shrink value) (basis value) e.g. flex: 1 1 100px;
  18.    */
  19.    flex-basis: 200px;
  20. }
  21.  
  22. .container-custom div:nth-child(2n + 1) {
  23.    background-color: darkblue;
  24.    flex-basis: 100px;
  25. }
