In this lesson, I was able to revisit on my knowledge for setting up a flexbox container. Before doing any flexbox styling, we are required to change the display property to flex. Like so: .container-custom{display: flex}

HTML
  1. <section class="container-custom"><!-- flex container -->
  2.     <div></div><!-- flex item -->
  3.     <div></div><!-- flex item -->
  4.     <div></div><!-- flex item -->
  5.     <div></div><!-- flex item -->
  6. </section>
CSS
  1. .container-custom {
  2.    display: flex;
  3. }
